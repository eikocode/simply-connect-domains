## Exhibit A: Licensed IP Documentation (DSR Algorithm)

**Algorithm Name:** Adaptive Spaced Repetition Engine (ASRE) v3.1  
**Core Variables:**  
1.  **Retrievability (R):** Probability of recall (0.0–1.0).
2.  **Stability (S):** Time in days for R to decay from 100% to 90%.
3.  **Difficulty (D):** Inherent complexity factor (1.0–10.0).

**Parameters:**  
- 19 distinct optimization parameters for calculating S and D.
- Default parameters derived from analysis of 220M+ review logs (Public Dataset Ref: KDD-2022-YE).

**Integration Requirements:**  
- Must support real-time updates based on optical sensor feedback (user position/mood).
- Must support data erasure triggers upon sync confirmation (Hybrid Storage Architecture).
