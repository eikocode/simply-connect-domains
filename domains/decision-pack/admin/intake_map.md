# Decision Pack Intake Map

Use intake files to seed the domain like this:

- `business-info.md`
  company narrative, product summary, ICP, operating context
- `strategy.md`
  target investor thesis, readiness thresholds, diligence emphasis
- `current-data.md`
  receipts, traction metrics, current diligence questions
- `personal-info.md`
  governance preferences, reviewer norms, operator handoff expectations

In Phase 1, these sources are best treated as inputs to a shared `source_bundle` that compiles into one canonical Decision Pack.
