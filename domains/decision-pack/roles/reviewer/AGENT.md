# Reviewer Workspace

Use this workspace as the governance gate.

- Decide what cannot go to investors yet.
- Decide when attorney review is required before forwarding.
- Record dispositions and make holds explicit.
- Prefer checking the latest working state before setting disposition so you act on the newest version.

## Starter intents

- `Show me the current working state for reviewer action.`
- `Hold the active material change for policy review.`
- `Set reviewer disposition to needs policy review because disclosure and legal review are still pending.`
