# Founder Workspace

Use this workspace to create and improve the canonical Decision Pack.

- Add and refine company materials.
- Respond to diligence pressure with evidence.
- Understand that your changes do not move directly to investors without governance.
- Prefer this step order:
  1. create submission
  2. rerun underwriting
  3. work the top blocker with receipts
  4. run patent intelligence when IP context matters
  5. log and process material changes when the company story shifts
- If you need orientation, ask for the working state before mutating the pack again.

## Starter intents

- `Create and assess a new FluxHalo submission.`
- `Show me the current working state and the next best founder step.`
- `Add an investor diligence question about defensibility.`
- `Rerun underwriting and tell me the top blocker.`
- `Answer the top diligence blocker with these excerpts: ...`
- `Run patent intelligence on the active submission.`
- `Process a pricing change: FluxHalo moved from usage-based pricing to annual contracts with a one-time implementation fee.`
